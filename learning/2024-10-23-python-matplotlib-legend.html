Tutorial: Positioning the Legend Outside of the Plot in Matplotlib

When plotting with Matplotlib, sometimes the legend can obstruct important parts of your plot. To avoid this, you can move the legend outside the plot area. This tutorial will guide you through several techniques for placing the legend outside of your plot.

1. Basic Plot with a Legend

Let’s start by creating a basic plot with two data series and a legend.

Example Plot:

import matplotlib.pyplot as plt

# Sample data
x = [1, 2, 3, 4]
y1 = [1, 4, 9, 16]
y2 = [1, 2, 3, 4]

# Create a plot
plt.plot(x, y1, label='y1')
plt.plot(x, y2, label='y2')

# Display the legend inside the plot
plt.legend()

plt.show()

Output:

In this case, the legend appears inside the plot, often covering part of your data. Let’s explore how to move it outside.

2. Moving the Legend to the Right of the Plot

You can move the legend to the right of the plot using bbox_to_anchor and loc. This allows the legend to appear outside the plot boundaries without overlapping the data.

Code:

import matplotlib.pyplot as plt

# Sample data
x = [1, 2, 3, 4]
y1 = [1, 4, 9, 16]
y2 = [1, 2, 3, 4]

# Create the plot
plt.plot(x, y1, label='y1')
plt.plot(x, y2, label='y2')

# Move the legend outside to the right
plt.legend(loc='center left', bbox_to_anchor=(1, 0.5))

# Show the plot
plt.show()

Explanation:

  • loc='center left': Positions the legend relative to the plot’s bounding box (the edges of the plot area).
  • bbox_to_anchor=(1, 0.5): Moves the legend outside the plot on the right. The first number (1) represents the x-coordinate (right edge of the plot), and 0.5 represents the vertical middle.

3. Placing the Legend Below the Plot

For some plots, especially those with long labels or many series, placing the legend below the plot may be better. Here’s how to do it:

Code:

import matplotlib.pyplot as plt

# Sample data
x = [1, 2, 3, 4]
y1 = [1, 4, 9, 16]
y2 = [1, 2, 3, 4]

# Create the plot
plt.plot(x, y1, label='y1')
plt.plot(x, y2, label='y2')

# Move the legend below the plot
plt.legend(loc='upper center', bbox_to_anchor=(0.5, -0.05), ncol=2)

# Show the plot
plt.show()

Explanation:

  • loc='upper center': Centers the legend at the top of the plot area.
  • bbox_to_anchor=(0.5, -0.05): Moves the legend below the plot (with a slight gap to prevent overlap).
  • ncol=2: Arranges the legend labels into two columns for a more compact display.

4. Adjusting the Plot Area to Fit the Legend

Sometimes, when you place the legend outside the plot, it may get clipped. To prevent this, you can adjust the layout of the plot.

Using subplots_adjust:

You can use plt.subplots_adjust() to expand the plot area and make space for the legend.

Code:

import matplotlib.pyplot as plt

# Sample data
x = [1, 2, 3, 4]
y1 = [1, 4, 9, 16]
y2 = [1, 2, 3, 4]

# Create the plot
plt.plot(x, y1, label='y1')
plt.plot(x, y2, label='y2')

# Move the legend outside to the right
plt.legend(loc='center left', bbox_to_anchor=(1, 0.5))

# Adjust the plot to make space for the legend
plt.subplots_adjust(right=0.75)  # Increase the right margin

plt.show()

Explanation:

  • plt.subplots_adjust(right=0.75): Expands the plot area, giving more space for the legend on the right.

5. Other Customizations for the Legend

You can further customize the legend in various ways:

  • Font Size: Adjust the font size for better readability using the fontsize parameter:

    plt.legend(fontsize='large')

  • Legend Title: Add a title to the legend with title:

    plt.legend(title='Legend Title')

  • Transparency: Make the legend box transparent by setting framealpha:

    plt.legend(framealpha=0.5)

Conclusion

This tutorial demonstrated how to place the legend outside the plot in Matplotlib, whether to the side or below the plot, and how to adjust the plot layout to accommodate the legend. These techniques will help you create clear, well-organized plots without any obstruction from the legend.

Summary of Key Steps:

  1. Use bbox_to_anchor to position the legend outside the plot.
  2. Use plt.subplots_adjust() to prevent clipping.
  3. Customize the legend’s appearance using parameters like fontsize, title, and framealpha.
