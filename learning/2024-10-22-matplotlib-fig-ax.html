Matplotlib: fig, ax, and subplots

In this tutorial, I’ll walk you through the basics of how to use matplotlib, with a focus on understanding fig, ax, and how to use subplots to create a figure with multiple visualizations. By the end, you’ll be comfortable creating visualizations with several subplots.

1. Understanding the Basics: fig and ax

When creating plots in matplotlib, there are two essential objects you should know:

  • fig (Figure): This is the whole figure or canvas. Think of it as the container that holds all your plots.
  • ax (Axes): These are the individual plots inside the figure. You can have multiple ax objects, each representing a subplot.

2. Using plt.subplots: Creating Multiple Plots in One Figure

To create subplots, matplotlib provides the plt.subplots() function, which allows you to create a grid of plots (or axes). Here’s how it works:

Syntax of plt.subplots()

fig, ax = plt.subplots(nrows, ncols, figsize)
  • nrows: The number of rows of subplots you want.
  • ncols: The number of columns of subplots you want.
  • figsize: The size of the overall figure (width, height in inches).

Example 1: Simple Grid of Subplots (2x2)

Let’s create a figure with a 2x2 grid of subplots, meaning 4 plots in total.

import matplotlib.pyplot as plt

# Create 2x2 grid of subplots
fig, ax = plt.subplots(2, 2, figsize=(8, 6))

# Access each subplot individually and plot some data
ax[0, 0].plot([1, 2, 3], [4, 5, 6])  # Top-left
ax[0, 1].plot([1, 2, 3], [6, 5, 4])  # Top-right
ax[1, 0].plot([1, 2, 3], [1, 2, 1])  # Bottom-left
ax[1, 1].plot([1, 2, 3], [3, 3, 3])  # Bottom-right

# Set titles for each subplot
ax[0, 0].set_title('Plot 1')
ax[0, 1].set_title('Plot 2')
ax[1, 0].set_title('Plot 3')
ax[1, 1].set_title('Plot 4')

# Adjust spacing between subplots
plt.tight_layout()

# Show the plot
plt.show()

In this example:

  • We create a 2x2 grid of subplots using plt.subplots(2, 2), so ax is a 2x2 array where each element is an individual Axes object.
  • You access each subplot using its grid position: ax[row, col].

3. Customizing Each Subplot

Each subplot (ax) can be customized individually. You can set titles, labels, and plot different types of graphs in each one. Here’s an example where we customize each subplot differently:

import matplotlib.pyplot as plt
import numpy as np

# Data for plots
x = np.linspace(0, 10, 100)

# Create 2x2 grid of subplots
fig, ax = plt.subplots(2, 2, figsize=(10, 8))

# Plot different types of data on each subplot
ax[0, 0].plot(x, np.sin(x), color='b', label='sin(x)')
ax[0, 0].set_title('Sine Wave')
ax[0, 0].legend()

ax[0, 1].plot(x, np.cos(x), color='r', label='cos(x)')
ax[0, 1].set_title('Cosine Wave')
ax[0, 1].legend()

ax[1, 0].scatter(x, np.sin(x), color='g', label='scatter')
ax[1, 0].set_title('Sine Scatter')
ax[1, 0].legend()

ax[1, 1].bar([1, 2, 3], [3, 2, 5], color='y', label='bar')
ax[1, 1].set_title('Bar Plot')
ax[1, 1].legend()

# Adjust layout so that subplots are spaced nicely
plt.tight_layout()

# Show the figure
plt.show()

4. Creating Complex Layouts (e.g., 4 or 6 subplots)

For more complex layouts, such as a 2x3 grid of subplots, you simply change the nrows and ncols arguments in plt.subplots().

Example 2: Creating a 2x3 Grid of Subplots (6 plots total)

import matplotlib.pyplot as plt
import numpy as np

# Data for plotting
x = np.linspace(0, 10, 100)

# Create a 2x3 grid of subplots
fig, ax = plt.subplots(2, 3, figsize=(12, 6))

# Add different plots to each subplot
for i in range(2):  # Loop over rows
    for j in range(3):  # Loop over columns
        ax[i, j].plot(x, np.sin(x + (i+j)), label=f'sin(x + {i+j})')
        ax[i, j].set_title(f'Plot {i * 3 + j + 1}')
        ax[i, j].legend()

# Adjust layout
plt.tight_layout()

# Show the figure
plt.show()

In this example:

  • We create a 2x3 grid using plt.subplots(2, 3).
  • You can still access each subplot with ax[row, col].

5. Shared Axes

Sometimes, you want to share the x-axis or y-axis between subplots. This can be helpful when the plots have the same x or y range. You can do this by using the sharex or sharey parameters in plt.subplots().

Example 3: Sharing Axes Between Subplots

import matplotlib.pyplot as plt
import numpy as np

# Data for plotting
x = np.linspace(0, 10, 100)
y1 = np.sin(x)
y2 = np.cos(x)

# Create subplots with shared y-axis
fig, ax = plt.subplots(2, 1, sharex=True, sharey=True, figsize=(8, 6))

# Plot on the two subplots
ax[0].plot(x, y1, label='sin(x)')
ax[0].set_title('Sine')
ax[0].legend()

ax[1].plot(x, y2, label='cos(x)')
ax[1].set_title('Cosine')
ax[1].legend()

# Label shared axes
ax[1].set_xlabel('x')
ax[0].set_ylabel('y')
ax[1].set_ylabel('y')

# Adjust layout
plt.tight_layout()

# Show the figure
plt.show()

Here:

  • sharex=True means that both subplots share the same x-axis.
  • sharey=True means that both subplots share the same y-axis.

6. Adding Titles, Labels, and Legends

Each subplot can have its own title, labels, and legends. You can set these using:

  • ax[i, j].set_title('Title'): Adds a title to the subplot.
  • ax[i, j].set_xlabel('X Label'): Adds a label to the x-axis.
  • ax[i, j].set_ylabel('Y Label'): Adds a label to the y-axis.
  • ax[i, j].legend(): Adds a legend to the subplot.

7. Adjusting Spacing Between Subplots

Sometimes, subplots can overlap or be too close to each other. To adjust the spacing, use:

  • plt.tight_layout(): Automatically adjusts the spacing between subplots to prevent overlaps.
  • fig.subplots_adjust(): Manually control the spacing.

Conclusion

Now you should have a solid understanding of how fig and ax work in matplotlib and how to create multiple subplots using plt.subplots(). Here’s a summary:

  • fig: The overall container for all subplots.
  • ax: Each individual subplot (Axes).
  • plt.subplots(): The easiest way to create multiple subplots.
  • Customize each subplot with titles, labels, and legends.
  • Adjust the layout using plt.tight_layout().
